# scraper

### Steps to run
```
pip install scrapy
cd scraper
scrapy crawl <spider-name>
```
> Replace <spider-name> with the one of the spider names from the spiders folder.

## 1. Wikihow scraper
A simple scrapper to scrape wikihow articles.

## 2. Amazon scraper
A simple scrapper to scrape amazon pages.
